* name: Informix
* website: (https://www.ibm.com/products/informix)
* Informix is a commercial database management developed and sold by the company of IBM. Informix is a relational data model that organizes and manages data and is an embedded hosting model. In order for Java or Python to access Informix, a JDBC driver as well as a ODBC driver must be used in order for the Java application to connect with the Informix database and the Python application to connect to the Informix database.
